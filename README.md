# Personal Portfolio

This is a modern, responsive personal portfolio built with React, Vite, and Tailwind CSS. It showcases my skills, projects, and provides a way for potential clients or employers to contact me.

## Features

- Responsive design that works on desktop and mobile
- Dark mode toggle
- Animated page transitions
- Project showcase with modal details
- Contact form with validation
- Timeline of professional journey
- Skills showcase
- Social media links

## Technologies Used

- React
- Vite
- Tailwind CSS
- Framer Motion for animations
- React Router for navigation
- Swiper for project carousel
- React Icons for icons
